Step 1 — Deploy the backend (once)
1. Go to script.google.com → New project
2. Paste Code.gs into the editor → Save
3. Deploy → New Deployment → Web App
4. Execute as: Me  ·  Access: Anyone
5. Authorise → copy the /exec URL

Step 2 — Host on GitHub Pages (once)
6. Go to github.com → New public repo
7. Upload this HTML file renamed as index.html
8. Settings → Pages → Branch: main → Save
9. Share https://yourname.github.io/repo with friends

Step 3 — Pre-configure (so friends skip this screen)
10. Open the GitHub Pages link → paste the /exec URL above → Connect
11. Open index.html in a text editor → find BAKED_IN_URL = ''
12. Paste your /exec URL inside the quotes → save → re-upload to GitHub
13. Friends open your GitHub Pages link → they go straight to Register ✓
